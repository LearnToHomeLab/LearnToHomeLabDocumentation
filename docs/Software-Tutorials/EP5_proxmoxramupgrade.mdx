1. Properly shut down VMs and containers in Proxmox.  
2. Ensure VMs and containers auto-boot after restart.  
3. Upgrade RAM on Proxmox nodes.  
4. Verify all services are back online.

## Container Auto Boot in Proxmox

Log into your Proxmox node or cluster and ensure **start at boot** is set to **yes** for containers.

<DocImage folder="images/ProxmoxRamUpgrade" name="image 1.png" alt="Container auto-start setting" />

## Virtual Machine Auto Boot in Proxmox

Similarly, set **start at boot** to **yes** for each virtual machine.

<DocImage folder="images/ProxmoxRamUpgrade" name="image 2.png" alt="VM auto-start setting" />

## Change Your RAM

Physically replace or upgrade the RAM on your Proxmox node, then power on the machine.

## Verify Services Are Online

Log back into Proxmox dashboard and confirm containers and VMs have green arrow indicators showing they are running.

<DocImage folder="images/ProxmoxRamUpgrade" name="image 3.png" alt="Confirm VMs running" />
