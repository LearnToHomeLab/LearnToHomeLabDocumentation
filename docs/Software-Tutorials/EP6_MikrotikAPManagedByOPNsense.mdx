## Configuring Mikrotik Wireless AP to Get DHCP Leases from OPNsense

1. Login to your Mikrotik AP using Ethernet connection to your PC using MAC address and login credentials from the device sticker.  
2. Open the **Bridge** tab.  
3. In **Bridge**, go to the **Ports** tab.  
4. Add all interfaces; typically add `ether1` to `bridge` if default config.  
5. In the **Interfaces** list, ensure:  
   - `LAN` is set to bridge.  
   - `WAN` is set to `ether1` (connected to internet).  
6. Open **DHCP Client** under **IP → DHCP Client**.  
7. Set the DHCP client on the `bridge` interface.  
8. Open **DHCP Server** under **IP → DHCP Server**.  
9. Disable the DHCP Server.  
10. Disable networks under the **Networks** tab in DHCP Server.  
11. Check **Address List** under **IP → Addresses** to verify your network settings are correct.  
12. Configure Wi-Fi settings under the **Wi-Fi** tab:  
   - Set SSID, band, channel width, passphrase, authentication type under **Configuration**, **Channel**, and **Security** tabs.
